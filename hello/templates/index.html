Optimus Rhyme

This is a project for Cornell CS4300 - Language and Information, which focuses on analyzing rhyming patterns and language used in poetry and rap.

Class Homepage Source on GitHub

Collaborators

  • Warren Crowell (wmc62)
  • Elly Nakahara (en254)

Project Goals

  • Rhyme completion based on rhyming scheme and sounds.
  • Line Generation based on research done in DopeLearning.
  • Incorporate user feedback to improve our similarity scoring.

Tools Used

  • Heroku
  • DopeLearning
  • RhymeBrain API
  • Genius.com API
